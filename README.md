# Testes de Regressão para PROCH (HLOOKUP) no LibreOffice Calc

Este repositório contém testes simples de regressão para a função PROCH (HLOOKUP) no LibreOffice Calc.

Foram criados dois casos de teste:
- Busca horizontal com valor encontrado
- Busca horizontal com valor inexistente (#N/D)

Os testes seguem o mesmo padrão de estrutura utilizado em testes de PROCV já existentes.
